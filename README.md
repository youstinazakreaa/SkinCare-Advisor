# 🌿 SkinCare Advisor

A Full-Stack web application that provides personalized skincare routines based on users' skin type, concerns, daily habits, and budget using a rule-based recommendation system.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 🧴 Personalized Skin Analysis
- 🌞 Morning & 🌙 Night Routine Generation
- 💰 Budget-Based Product Recommendations
- ⚠️ Product Conflict Detection
- 👨‍💼 Admin Dashboard
- 📱 Responsive Design

---

## 🛠️ Technologies Used

### Frontend
- Angular
- TypeScript
- HTML
- CSS
- Tailwind CSS

### Backend
- ASP.NET Core Web API
- C#
- Entity Framework Core

### Database
- SQL Server
